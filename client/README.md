# Recipe 2 Grocery List - Frontend

Simple app to store recipes and create grocery list from the ingredients of selected recipes.

![VueJS](https://img.shields.io/badge/vue-3.2.x-green.svg)
![Vite](https://img.shields.io/badge/vite-4.1.x-blue.svg)
![Axios](https://img.shields.io/badge/axios-1.3.x-lightgrey.svg)

## Config

Uses `dotenv` to read env variables
- `VITE_BACKEND_URL` - backend endpoint to read/update DB

```dotenv
VITE_BACKEND_URL=http://localhost:8080
```
